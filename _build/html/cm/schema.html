class cm.models.Consent_Request(**kwargs)¶

Details of a Consent Request.

Parameters

  • id (int) – primary key

  • request_id (int) – ID of the consent request at the HIU

  • user_id (int) – foreign key to User ID of the user (patient)

  • hiu_id (int) – ID of the HIU

  • hiu_name (str) – Name of the HIU

  • requester_name (str) – Name of the user requesting for consent

  • hip_id (int) – ID of the HIP

  • hip_name (str) – Name of the HIP

  • encounter_id (int) – nullable ID of the Encounter

  • record_id (int) – nullable ID of the Record

  • purpose (Enum(utils.PurposeType)) – Purpose of the request

  • time_from (DateTime) – Time from which the consent is valid

  • time_to (DateTime) – Time till which the consent is valid

  • status (Enum(utils.StatusType)) – Status of the request
