API¶

gateway.routes.consent_listener()¶
POST /consent_listener/¶

Pass a consent status and object to a HIU.

Example request:

POST /consent_listener/ HTTP/1.1
Host: http://127.0.0.1:5000/
Content-Type: application/json

{
    "consent_id": 1,
    "hiu_id": 1,
    "artefact":
        {
            "hiu_id": 1,
            "hip_id": 1,
            "purpose": "Diagnosis",
            "time_from": "2020-11-21",
            "time_to": "2020-11-23",
            "encounter_id": 1
        },
    "signature": ".....",
    "accept": true
}
Request JSON Object

  • consent_id (int) – ID of the consent request at the HIU.

  • hiu_id (int) – ID of the HIU.

  • artefact (json) – Consent Artefact.

  • signature (string) – Digital signature of the artefact.

  • accept (bool) – Status of the consent request.

Example response:

HTTP/1.1 201 Received Request
Status Codes

  • 201 Created – Received Request

  • 400 Bad Request – HIU not found

gateway.routes.consent_request()¶
POST /consent_request/¶

Pass a consent request to the Consent Manager.

Example request:

POST /consent_request/ HTTP/1.1
Host: http://127.0.0.1:5000/
Content-Type: application/json

{
    "request_id": 1,
    "health_id": "1",
    "hiu_id": 1,
    "requester_name": "Doctor1",
    "hip_id": 2,
    "purpose": "Diagnosis",
    "time_from": "2020-11-21",
    "time_to": "2020-11-23",
    "encounter_id": 1,
}
Request JSON Object

  • request_id (int) – ID of the consent request at the HIU.

  • health_id (string) – Health ID of the user (patient).

  • hiu_id (int) – ID of the HIU.

  • requester_name (string) – Name of the user requesting the consent.

  • hip_id (int) – ID of the HIP.

  • purpose (string) – Purpose of the consent request.

  • time_from (string) – Date from which the consent is valid.

  • time_to (string) – Date till which the consent is valid.

  • encounter_id (int) – optional ID of the encounter at the HIP.

  • record_id (int) – optional ID of the record at the HIP.

Example response:

HTTP/1.1 201 Received Request
Status Codes

  • 201 Created – Received Request

  • 400 Bad Request – User not found

  • 400 Bad Request – HIU not found

  • 400 Bad Request – HIP not found
